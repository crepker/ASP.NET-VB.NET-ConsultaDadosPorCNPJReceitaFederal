# ASP.NET-VB.NET-ConsultaDadosPorCNPJReceitaFederal
Pequena aplicação para pesquisar os dados de uma empresa a partir do CNPJ. Os dados são pesquisados no site da receita federal.
Procurei muito na internet por uma solução para este problema em VB.NET, mas não encontrei nenhuma que fosse satisfatória, por isso resolvi juntar as pequenas partes que fui encontrando e consolidar uma solução nesta linguagem.
Aproveitei para desenvolver uma validação do CNPJ em javascript, assim, antes que o acesso ao site da receita federal ocorra (o que pode levar algum tempo para ser concluído) já acontece uma pré checagem do valor inserido.
Espero que seja útil.
